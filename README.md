# Vibe Coding — 用对话创造软件

一个介绍 **Vibe Coding** 概念的现代化单页网站——不写一行代码，用自然语言描述你的想法，AI 即刻生成可运行的软件。

## 功能

- **对话演示**：模拟真实的 Vibe Coding 交互过程，展示从自然语言到代码的完整流程
- **核心概念讲解**：阐述 Vibe Coding 的三个核心原则——描述意图、AI 生成代码、像聊天一样迭代
- **传统编程对比**：直观对比传统编程与 Vibe Coding 的工作流程差异
- **四步上手教程**：从描述需求到对话式迭代的完整指引
- **工具推荐**：内置 Claude Code、Cursor、v0 等主流工具介绍
- **交互沙箱**：用户可直接在页面中输入需求，体验 AI 辅助编码的过程
- **响应式设计**：适配桌面端与移动端，暗色主题贯穿始终

## 技术栈

- 纯静态 HTML/CSS/JavaScript，无需构建工具
- 自定义 CSS 变量体系，统一暗色主题
- 原生 JS 实现对话动画、沙箱交互、滚动效果

## 本地运行

直接用浏览器打开 `index.html` 即可：

```bash
open index.html
```

或使用任意静态服务器：

```bash
npx serve .
```
